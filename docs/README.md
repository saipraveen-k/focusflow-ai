# FocusFlow AI 🚀

## 🧠 Overview

FocusFlow AI is a smart notification filtering system that reduces distractions by analyzing user context and prioritizing relevant notifications.

## ✨ Features
* Context-aware filtering
* AI-based decision making
* Smart scheduling
* User behavior learning
* Transparency dashboard

## 🛠️ Tech Stack
* React Native
* Node.js
* Python (ML)
* MongoDB

## 📊 Use Cases
* Students during study time
* Professionals in meetings
* Drivers avoiding distractions

## 🚀 Future Scope
* NLP for message understanding
* Cross-device sync
* Voice-based controls
